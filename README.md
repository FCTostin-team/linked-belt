A specialized web utility for generating Lua console commands to create and connect "Linked Belts" in Factorio. These hidden entities allow for instant item transport over any distance, and this tool simplifies the process of spawning and linking them via unique tags.

## Features

* **Code Generation**: Automatically generates the necessary `/c` Lua commands to spawn linked belts at the player's position.
* **Tag Management**: Allows custom naming (tags) for input and output belts to ensure correct pairing.
* **Connection Script**: Generates the specific logic to link the two entities (`connect_linked_belts`) based on the assigned tags.
* **Factorio UI**: Designed with a dark, pixel-art interface mimicking the game's aesthetic.

## Installation and Usage

1.  Clone the repository or download the `index.html` file.
2.  Open the file in any web browser.
3.  Enter unique names for your "Input" and "Output" belts.
4.  Copy the generated commands and paste them into the Factorio console (opened with `~`).
